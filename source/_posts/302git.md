---
title: 如何解决启用SteamCommunity302后git命令行无法连接
date: 2025-03-16 20:23:43
tags: 疑难解答
---

众所周知国内对GitHub的访问非常不稳定, 大部分稳定访问方法都需要借助非常规手段, 其中比较靠谱且免费的是SteamCommunity302, 不仅能加速Steam商店, 还能稳定访问GitHub. 然而, 启动SteamCommunity302的时候, 虽然浏览器能流畅访问GitHub, 但是命令行中的git会连接失败, 报错类似

```
SSL peer certificate or SSH remote key was not OK
```

解决方案如下:

方法一(推荐):

找到SteamCommunity302根目录下的```steamcommunity.crt```证书文件, 双击打开, 点击安装证书-下一步-将所有证书放入下列存储-浏览, 选择"受信任的根证书颁发机构", 点击安装即可.

方法二(不推荐):

终端运行:

```bash
git config --global http.sslVerify false
```

暂时关闭git的SSL安全验证, 但存在安全隐患, 不建议长期关闭.
