把马士兵的 PHP 跳着看完了, 大概学了点 SQL, 后面的 MySQL 实在不太想看了, 直接开始看渗透了.

反正先看着吧, 这些东西估计迟早要学的.

今天开始看信息收集, 在 iPad 上下了个 Markdown 笔记软件, 认真照着网课记笔记, 记了一堆常用端口, 记到一半开始反思, 我记这些完全没屁用, 反正要用的时候直接查就行了啊. 笑话小红书五颜六色的勾股定理笔记笑话了那么久, 最后自己也走上了假努力和自我催眠的道路. 不过 Markdown 用着确实舒服.

前几天听说 VMWare Workstation 对个人用户完全免费了, 今天发现 VirtualBox 复制粘贴又有毛病, 直接卸了准备换 VMWare 了. VMWare 的正版网站藏得还挺深的, 必应也是广告一堆.

今天又看了八集芙莉莲, 很好看.

估算了一下, 网课进度好像有点太慢了, 同学已经在看逆向了. 目前难度倒没那么夸张, 但是占据了业余爱好的时间真的有点难受. 缺的还是时间, 从小到大缺的都是时间. 他妈的.

电磁 C 终于拿了个 3.7, 擦边拿了个 A-, 不容易啊.

明天继续自习.
